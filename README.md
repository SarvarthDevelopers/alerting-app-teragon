# Teragon - Manufacturing Quality Monitoring

A high-performance, responsive mobile user interface for **Teragon**, a steel quality monitoring and alerting application, meticulously crafted to provide a premium user experience.

## Getting Started

To get the development environment up and running, follow these steps:

1. **Install Dependencies**:

   ```bash
   npm install
   ```
2. **Start Development Server**:

   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Core**: React + TypeScript
- **Styling**: Tailwind CSS
- **Components**: Radix UI, Lucide React
- **Animations**: Motion
- **Build Tool**: Vite
